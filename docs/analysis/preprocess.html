Preprocessing IPUMS fertility data

Author

Tina Lasisi

Published

June 29, 2026

This notebook converts a restricted IPUMS USA extract into the aggregate fertility summaries used throughout the kincounts project, and documents the processing decisions. It makes two analytic contributions:

  1. Completed-fertility distributions by birth cohort — women aged 50–59 in each census from 1950 to 1990 (children ever born), the inputs used by the downstream fertility model and simulator.
  2. A common-universe analysis — a measurement of how those distributions change when the population that was asked the children-ever-born question shifts from ever-married women (1950–1960) to all women (1970–1990).

Because IPUMS USA policy restricts redistribution of raw microdata, only the aggregate CSVs in data/processed/ are committed; the raw extract stays local-only (ignored by Git via .gitignore). Processing code is collapsed by default; the sections below present the resulting figures and tables.

1 Data source and access

The source data are an IPUMS USA extract (a fixed-width *.dat.gz plus its *.xml DDI codebook) covering the 1950, 1960, 1970, 1980, and 1990 U.S. Census samples. The variables used are YEAR, SAMPLE, PERWT, SEX, AGE, BIRTHYR, MARST, GQ, and CHBORN. The sample-line weight SLWT, the allocation flag QCHBORN, and the nativity/birthplace variables NATIVITY/BPL were not requested; their absence is discussed in the limitations.

Formal citation:

Steven Ruggles, Sarah Flood, Matthew Sobek, Daniel Backman, Grace Cooper, Julia A. Rivera Drew, Stephanie Richards, Renae Rodgers, Jonathan Schroeder, and Kari C.W. Williams. IPUMS USA: Version 16.0 [dataset]. Minneapolis, MN: IPUMS, 2025. https://doi.org/10.18128/D010.V16.0

To reproduce the raw-data step, create your own IPUMS USA extract with the same samples and variables, place the .dat.gz and .xml files in data/raw/, and render this notebook locally. Only the aggregate CSVs in data/processed/ should be committed.

2 Data processing

The code below locates the extract, reads it through its DDI codebook, and builds the aggregates presented in the following sections. It runs only when the raw extract is present and the cached outputs are out of date; otherwise the committed CSVs are loaded directly.

if (rebuild_processed) {
  # Parse only the variables used for filtering, cohort assignment, weights,
  # and fertility counts.
  ipums_data <- read_ipums_micro(
    ddi,
    vars = analysis_vars,
    data_file = dat_gz_file,
    verbose = FALSE
  ) %>%
    mutate(across(all_of(analysis_vars), as.numeric))

  # --- Main series: age-first cohort assignment (membership defined by AGE within
  # a census). The 1950 cohort is a proxy from women aged 60-69 in 1960; direct
  # 1950 (women 50-59 in 1950) is held out for validation. Birth-year spans are
  # derived afterwards for labeling only.
  cohort_records <- ipums_data %>%
    filter(SEX == 2) %>%
    mutate(
      source_year = YEAR,
      cohort_source = case_when(
        YEAR == 1950 & between(AGE, 50, 59) ~ "direct_1950_validation",
        YEAR == 1960 & between(AGE, 60, 69) ~ "proxy_1950",
        YEAR == 1960 & between(AGE, 50, 59) ~ "main_1960",
        YEAR == 1970 & between(AGE, 50, 59) ~ "main_1970",
        YEAR == 1980 & between(AGE, 50, 59) ~ "main_1980",
        YEAR == 1990 & between(AGE, 50, 59) ~ "main_1990",
        TRUE ~ NA_character_
      ),
      Year = case_when(
        cohort_source %in% c("proxy_1950", "direct_1950_validation") ~ 1950,
        cohort_source == "main_1960" ~ 1960,
        cohort_source == "main_1970" ~ 1970,
        cohort_source == "main_1980" ~ 1980,
        cohort_source == "main_1990" ~ 1990,
        TRUE ~ NA_real_
      ),
      series = case_when(
        cohort_source == "direct_1950_validation" ~ "validation",
        !is.na(cohort_source) ~ "main",
        TRUE ~ NA_character_
      ),
      valid_chborn = CHBORN > 0 & CHBORN < 97,
      n_child = if_else(valid_chborn, pmin(CHBORN - 1, 12), NA_real_)
    ) %>%
    filter(!is.na(cohort_source))

  cohort_composition <- cohort_records %>%
    group_by(series, cohort_source, source_year, Year) %>%
    summarise(
      n_records = n(),
      age_min = min(AGE, na.rm = TRUE),
      age_max = max(AGE, na.rm = TRUE),
      birthyr_min = min(BIRTHYR[BIRTHYR > 1800], na.rm = TRUE),
      birthyr_max = max(BIRTHYR[BIRTHYR > 1800], na.rm = TRUE),
      .groups = "drop"
    ) %>%
    mutate(cohort = paste0(birthyr_min, "-", birthyr_max)) %>%
    arrange(Year, source_year, cohort_source)

  cohort_records <- cohort_records %>%
    left_join(select(cohort_composition, cohort_source, cohort), by = "cohort_source")

  age_cohort_validation <- cohort_composition
  completed_fertility <- cohort_records
  valid_completed_fertility <- completed_fertility %>% filter(valid_chborn)
  main_fertility <- valid_completed_fertility %>% filter(series == "main")
  validation_1950 <- valid_completed_fertility %>%
    filter(cohort_source %in% c("proxy_1950", "direct_1950_validation"))

  # How much data the CHBORN validity rule excludes, by cohort source.
  chborn_universe <- completed_fertility %>%
    count(cohort_source, source_year, Year, cohort, valid_chborn, name = "n_records") %>%
    pivot_wider(
      names_from = valid_chborn,
      values_from = n_records,
      names_prefix = "valid_chborn_",
      values_fill = 0
    ) %>%
    mutate(
      total_records = valid_chborn_FALSE + valid_chborn_TRUE,
      valid_share = valid_chborn_TRUE / total_records
    ) %>%
    arrange(Year, source_year, cohort_source)

  cohort_means <- valid_completed_fertility %>%
    group_by(cohort_source, series, source_year, Year, cohort) %>%
    summarise(
      n_women = n(),
      n_women_weighted = sum(PERWT, na.rm = TRUE),
      mean_children = mean(n_child),
      mean_children_weighted = weighted.mean(n_child, PERWT, na.rm = TRUE),
      share_12plus = mean(n_child == 12),
      .groups = "drop"
    ) %>%
    arrange(series, Year, source_year, cohort_source)

  fertility_counts <- main_fertility %>%
    count(Year, cohort, n_child, name = "n_women") %>%
    arrange(Year, n_child)

  fertility_counts_weighted <- main_fertility %>%
    group_by(Year, cohort, n_child) %>%
    summarise(n_women_weighted = sum(PERWT, na.rm = TRUE), .groups = "drop") %>%
    arrange(Year, n_child)

  fertility_pmf <- fertility_counts %>%
    group_by(Year, cohort) %>%
    mutate(probability = n_women / sum(n_women)) %>%
    ungroup() %>%
    transmute(Year, cohort, children = n_child, probability)

  fertility_pmf_weighted <- fertility_counts_weighted %>%
    group_by(Year, cohort) %>%
    mutate(probability_weighted = n_women_weighted / sum(n_women_weighted)) %>%
    ungroup() %>%
    transmute(Year, cohort, children = n_child, probability_weighted)

  cohort_sample_sizes <- main_fertility %>%
    group_by(Year, cohort) %>%
    summarise(
      n_women = n(),
      n_women_weighted = sum(PERWT, na.rm = TRUE),
      .groups = "drop"
    ) %>%
    arrange(Year)

  validation_1950_summary <- validation_1950 %>%
    group_by(cohort_source) %>%
    summarise(
      source_year = first(source_year),
      output_year = first(Year),
      n_women = n(),
      n_women_weighted = sum(PERWT, na.rm = TRUE),
      mean_children = mean(n_child),
      mean_children_weighted = weighted.mean(n_child, PERWT, na.rm = TRUE),
      share_12plus = mean(n_child == 12),
      .groups = "drop"
    ) %>%
    arrange(cohort_source)

  validation_1950_pmf <- validation_1950 %>%
    count(cohort_source, n_child, name = "n_women") %>%
    left_join(
      validation_1950 %>%
        group_by(cohort_source, n_child) %>%
        summarise(n_women_weighted = sum(PERWT, na.rm = TRUE), .groups = "drop"),
      by = c("cohort_source", "n_child")
    ) %>%
    group_by(cohort_source) %>%
    mutate(
      probability = n_women / sum(n_women),
      probability_weighted = n_women_weighted / sum(n_women_weighted)
    ) %>%
    ungroup() %>%
    arrange(cohort_source, n_child)

  # --- Common-universe analysis (separate frame; the main series is untouched).
  # Follow decade birth cohorts across the 1960-1990 censuses and, where the whole
  # cohort has reached age 50+, contrast the children-ever-born distribution under
  # two universes: all women vs ever-married. This isolates the effect of the
  # universe shift (1950/1960 asked only ever-married women; 1970-1990 all women).
  # MARST: 1-5 = ever-married, 6 = never-married, 9 = blank/missing (kept in the
  # all-women universe but excluded from the ever-married subset).
  band_records <- ipums_data %>%
    filter(SEX == 2, between(YEAR, 1960, 1990), between(BIRTHYR, 1890, 1939)) %>%
    mutate(
      band_start = 10L * (BIRTHYR %/% 10L),
      birth_band = paste0(band_start, "-", band_start + 9L),
      universe_year = YEAR,
      # Keep only cells where the whole decade cohort has reached age 50+.
      cohort_complete = (universe_year - (band_start + 9L)) >= 50L,
      valid_chborn = CHBORN > 0 & CHBORN < 97,
      n_child = if_else(valid_chborn, pmin(CHBORN - 1, 12), NA_real_),
      marital = case_when(
        MARST %in% 1:5 ~ "ever_married",
        MARST == 6 ~ "never_married",
        TRUE ~ "unknown"
      ),
      gq_household = GQ %in% c(1, 2, 5)
    ) %>%
    filter(cohort_complete, valid_chborn)

  band_long <- bind_rows(
    band_records %>% mutate(universe = "all_women"),
    band_records %>% filter(marital == "ever_married") %>% mutate(universe = "ever_married")
  )

  band_summary <- band_long %>%
    group_by(birth_band, universe_year, universe) %>%
    summarise(
      age_min = min(AGE),
      age_max = max(AGE),
      n_women = n(),
      n_women_weighted = sum(PERWT),
      mean_children = mean(n_child),
      mean_children_weighted = weighted.mean(n_child, PERWT),
      var_children = var(n_child),
      share_0 = mean(n_child == 0),
      share_12plus = mean(n_child == 12),
      gq_household_share = mean(gq_household),
      .groups = "drop"
    ) %>%
    arrange(birth_band, universe_year, universe)

  band_pmf <- band_long %>%
    count(birth_band, universe_year, universe, n_child, name = "n_women") %>%
    left_join(
      band_long %>%
        group_by(birth_band, universe_year, universe, n_child) %>%
        summarise(n_women_weighted = sum(PERWT), .groups = "drop"),
      by = c("birth_band", "universe_year", "universe", "n_child")
    ) %>%
    group_by(birth_band, universe_year, universe) %>%
    mutate(
      probability = n_women / sum(n_women),
      probability_weighted = n_women_weighted / sum(n_women_weighted)
    ) %>%
    ungroup() %>%
    arrange(birth_band, universe_year, universe, n_child)

  # Within-census contrast. The two universes coincide by construction before 1970
  # (never-married women were not in the CHBORN universe), so the contrast is
  # meaningful only for universe_year >= 1970.
  universe_contrast <- band_summary %>%
    select(birth_band, universe_year, universe, n_women, mean_children, var_children, share_0) %>%
    pivot_wider(
      names_from = universe,
      values_from = c(n_women, mean_children, var_children, share_0)
    ) %>%
    mutate(
      mean_shift = mean_children_all_women - mean_children_ever_married,
      share0_shift = share_0_all_women - share_0_ever_married,
      pct_never_married = 1 - n_women_ever_married / n_women_all_women
    )

  tvd_table <- band_pmf %>%
    select(birth_band, universe_year, universe, n_child, probability, probability_weighted) %>%
    pivot_wider(
      names_from = universe,
      values_from = c(probability, probability_weighted),
      values_fill = 0
    ) %>%
    group_by(birth_band, universe_year) %>%
    summarise(
      tvd = 0.5 * sum(abs(probability_all_women - probability_ever_married)),
      tvd_weighted = 0.5 * sum(abs(probability_weighted_all_women - probability_weighted_ever_married)),
      .groups = "drop"
    )

  universe_contrast <- universe_contrast %>%
    left_join(tvd_table, by = c("birth_band", "universe_year")) %>%
    arrange(birth_band, universe_year)

  # Early-cohort decomposition: the two cohorts seen ever-married at completed
  # fertility (1960) and all-women later (1970). Split the gap into the universe
  # effect (same census, adding never-married women) and the mortality/decade
  # effect (same ever-married universe, older age).
  pick_mean <- function(b, y, u) {
    v <- band_summary$mean_children[
      band_summary$birth_band == b &
        band_summary$universe_year == y &
        band_summary$universe == u
    ]
    if (length(v) == 1) v else NA_real_
  }
  early_cohort_followup <- tibble(
    birth_band = c("1890-1899", "1900-1909"),
    completed_year = c(1960, 1960),
    later_year = c(1970, 1970)
  ) %>%
    mutate(
      completed_em_mean = mapply(function(b, y) pick_mean(b, y, "ever_married"), birth_band, completed_year),
      later_em_mean = mapply(function(b, y) pick_mean(b, y, "ever_married"), birth_band, later_year),
      later_all_mean = mapply(function(b, y) pick_mean(b, y, "all_women"), birth_band, later_year),
      universe_effect = later_all_mean - later_em_mean,
      mortality_effect = later_em_mean - completed_em_mean
    )

  # Self-check: the 1900-1909 / 1960 ever-married mean should be close to the main
  # 1960 series mean, and 1890-1899 / 1960 close to the proxy-1950 mean.
  message(
    "Self-check (common-universe vs main series): ",
    "1900-1909/1960 ever-married mean = ", round(pick_mean("1900-1909", 1960, "ever_married"), 3),
    " (main_1960 ~ 2.429); ",
    "1890-1899/1960 ever-married mean = ", round(pick_mean("1890-1899", 1960, "ever_married"), 3),
    " (proxy_1950 ~ 2.818)."
  )

  # Write the aggregate CSVs.
  write_csv(cohort_sample_sizes, file.path(processed_dir, "ipums_cohort_sample_sizes.csv"))
  write_csv(fertility_counts, file.path(processed_dir, "ipums_fertility_counts_by_cohort.csv"))
  write_csv(fertility_pmf, file.path(processed_dir, "ipums_fertility_pmf_by_cohort.csv"))
  write_csv(fertility_counts_weighted, file.path(processed_dir, "ipums_fertility_counts_by_cohort_weighted.csv"))
  write_csv(fertility_pmf_weighted, file.path(processed_dir, "ipums_fertility_pmf_by_cohort_weighted.csv"))
  write_csv(validation_1950_summary, file.path(processed_dir, "validation_1950_direct_vs_proxy.csv"))
  write_csv(validation_1950_pmf, file.path(processed_dir, "validation_1950_direct_vs_proxy_pmf.csv"))
  write_csv(age_cohort_validation, file.path(processed_dir, "ipums_age_cohort_validation.csv"))
  write_csv(chborn_universe, file.path(processed_dir, "ipums_chborn_universe_by_cohort.csv"))
  write_csv(cohort_means, file.path(processed_dir, "ipums_cohort_fertility_checks.csv"))
  write_csv(band_summary, file.path(processed_dir, "ipums_cohort_universe_summary.csv"))
  write_csv(band_pmf, file.path(processed_dir, "ipums_cohort_universe_pmf.csv"))
  write_csv(universe_contrast, file.path(processed_dir, "ipums_universe_contrast_by_cohort.csv"))
  write_csv(early_cohort_followup, file.path(processed_dir, "ipums_early_cohort_followup.csv"))
  write_csv(
    tibble(
      processing_version = processing_version,
      cohort_definition = "main series: age-defined cohorts (50-59; proxy-1950 from 60-69 in 1960; direct-1950 validation-only); CHBORN harmonized to 12+. Common-universe analysis: decade birth cohorts followed across 1960-1990 at completed ages, all-women vs ever-married (MARST 1-5).",
      rebuilt_at = as.character(Sys.time())
    ),
    file.path(processed_dir, "ipums_preprocess_metadata.csv")
  )

  fertility_pmf %>%
    group_split(Year) %>%
    lapply(function(dat) {
      year <- unique(dat$Year)
      write_csv(
        dat %>% transmute(children, probability),
        file.path(processed_dir, paste0("fertility_pmf_", year, ".csv"))
      )
    })

  rm(
    ipums_data, cohort_records, cohort_composition, completed_fertility,
    valid_completed_fertility, main_fertility, validation_1950,
    band_records, band_long, tvd_table
  )
} else {
  # Load the committed aggregates without reparsing restricted person-level data.
  rd <- function(f) read_csv(file.path(processed_dir, f), show_col_types = FALSE)
  cohort_sample_sizes <- rd("ipums_cohort_sample_sizes.csv")
  fertility_counts <- rd("ipums_fertility_counts_by_cohort.csv")
  fertility_pmf <- rd("ipums_fertility_pmf_by_cohort.csv")
  fertility_counts_weighted <- rd("ipums_fertility_counts_by_cohort_weighted.csv")
  fertility_pmf_weighted <- rd("ipums_fertility_pmf_by_cohort_weighted.csv")
  validation_1950_summary <- rd("validation_1950_direct_vs_proxy.csv")
  validation_1950_pmf <- rd("validation_1950_direct_vs_proxy_pmf.csv")
  age_cohort_validation <- rd("ipums_age_cohort_validation.csv")
  chborn_universe <- rd("ipums_chborn_universe_by_cohort.csv")
  cohort_means <- rd("ipums_cohort_fertility_checks.csv")
  band_summary <- rd("ipums_cohort_universe_summary.csv")
  band_pmf <- rd("ipums_cohort_universe_pmf.csv")
  universe_contrast <- rd("ipums_universe_contrast_by_cohort.csv")
  early_cohort_followup <- rd("ipums_early_cohort_followup.csv")
  preprocess_metadata <- rd("ipums_preprocess_metadata.csv")

3 The children-ever-born question and its universe

CHBORN records the number of children ever born alive to a woman. It counts live births only — stillbirths are excluded, as are adopted children and stepchildren.

For a multi-decade comparison, the central complication is that who was asked the question — the variable’s universe — changed across censuses:

Census Universe (who was asked)
1950 ever-married women, and only the 1-in-6 sample line
1960 ever-married women
1970 all women age 15+
1980 all women age 15+
1990 all women age 15+

A woman not in the universe was not asked the question and has no recorded value (IPUMS code 00), which is distinct from a recorded count of zero. Such records are excluded from all fertility distributions. Two consequences follow.

Consequence 1 — coverage (1950). Because 1950 asked only ever-married sample-line women, most 1950 women have no CHBORN value. The figure below shows, for each census/age group, the share of women who were asked the question: direct-1950 is near 30% (only ever-married sample-line women); the 1960 census — the source of both the 1960 cohort and the 1950 proxy — is near 92% (ever-married women only); and the 1970–1990 censuses are at 100% (all women). This is why the 1950 cohort is taken from a proxy (the same birth cohort observed in 1960) rather than directly (see Defining the analytic sample).

Consequence 2 — the universe shift (comparability). The early cohorts describe ever-married women only; the later cohorts describe all women. A raw cross-cohort series therefore mixes two populations: the 1970–1990 samples include never-married (largely childless) women that the 1950 and 1960 samples cannot. The only universe common to every census is ever-married women, which can be reconstructed in any year from marital status (MARST); this motivates the common-universe analysis below.

4 Defining the analytic sample

The analytic population is women (SEX == 2) observed at completed fertility. Cohort membership for the main series is defined by age within a census — women aged 50–59 — not by birth year, so nobody in the age window is dropped because their birthday had not yet occurred by the April-1 census date. Birth-year spans are derived from the data afterwards, for labeling only.

5 CHBORN coding and harmonization

ipumsr returns the raw IPUMS codes. Code 00 is not applicable (not in universe — not zero children); zero children is code 01, one child 02, and so on. Per the IPUMS user note, children ever born is the code minus one, after dropping 00. We keep codes 01–61, set n_child = CHBORN − 1, and cap at 12 (12+) so all cohorts share one top category: the 1960–1990 samples top-code at 12+ (code 13), while 1950 resolves higher (to 25+, code 26).

6 Putting cohorts on a common universe

We treat fertility as effectively complete by age 50; the 50–59 age window is chosen to capture completed fertility while limiting the mortality selection that affects older ages. Under this assumption, differences in a fixed birth cohort’s measured children-ever-born distribution across successive censuses reflect measurement — mortality selection, migration, response error, and universe or top-code differences — rather than a change in fertility. We therefore estimate the universe effect by comparing the two universes within a single census, and use the cross-census comparison only to gauge the magnitude of these measurement differences.

We follow each decade birth cohort across the 1960–1990 censuses, restricting to cells where the whole cohort has reached age 50+, and within each census compute the children-ever-born distribution two ways: all women vs ever-married (MARST 1–5). Before 1970 the two coincide by construction (never-married women were not in the universe), so the contrast is informative only for 1970 onward.

Interpretation. The within-census all-women vs ever-married contrast (mean_shift, tvd, and the 1970 PMFs) is the preferred estimate: it compares the same survivors in the same census, with appropriate PERWT weighting. Differences across censuses — the drift along each trajectory — bound the combined measurement difference rather than a change in fertility. Key caveats:

  • Mortality selection. Never-married women have higher old-age mortality, so the all-women pool drifts toward ever-married at older ages — the universe gap is attenuated at older ages. The least-contaminated estimate is the contrast at the youngest census in which a cohort is observed under the all-women universe.
  • Never-married fertility is a lower bound. Historically nonmarital births are under-reported, so the all-women mean understates true all-women fertility and the gap is, if anything, overstated.
  • Weighting. PERWT is appropriate for 1970–1990 (CHBORN is universal there), so the within-census contrast is unaffected by the weighting issue below. 1950 is excluded from this analysis (its sample-line weight SLWT is absent), and the 1960 cells are ever-married by construction.
  • Migration / nativity. Birth cohorts are not closed across censuses; NATIVITY/BPL are absent, so cross-census comparisons cannot be restricted to the native-born — a further reason to rely on the within-census contrast, which compares a single population. gq_household_share is reported per cell as a coarse check on institutionalization at older ages.

In sum, the pre-1970 cohorts (1950 and 1960) are observed only for ever-married women, which raises their measured completed fertility by roughly 0.1–0.2 children relative to an all-women basis. This gap is small next to the Depression-era decline in completed fertility across the 1901–1920 birth cohorts, and it is reported as a sensitivity rather than applied to the main series.

9 Limitations

  • 12+ top-code. All outputs harmonize the child-count support to 0, 1, …, 11, 12+. 1960–1990 already top-code at 12+, but 1950 resolves to 25+, so the cap discards tail detail for the (validation-only) 1950 series and for older cohorts seen in 1950.
  • Weighting. Main outputs are unweighted; PERWT-weighted versions are provided as a sensitivity. The 1950 sample-line weight SLWT is absent, so any 1950 weighted figure is approximate; 1950 is therefore kept out of the weighted cross-cohort series.
  • Extract gaps. A future extract should add SLWT (correct 1950 weighting), NATIVITY/BPL (restrict to native-born / measure cohort compositional drift across censuses), and QCHBORN (filter allocated/imputed values). Each would close a caveat that the current data cannot resolve.
  • Sharing. The restricted IPUMS records, person-level weights, and raw fixed-width data remain local-only and are not committed; only the aggregate CSVs are.
